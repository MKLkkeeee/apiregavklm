REST API Reference

Use these endpoints to read and write data from your applications (Discord Bots, Game Servers, etc.)

GET /api/data

อ่านข้อมูล JSON ทั้งหมดจาก Database ของคุณ

Headers

x-api-key: "nx_your_api_key_here"

Example Output (Bot Node.js)

const res = await fetch('http://localhost:3000/api/data', {
    headers: { 'x-api-key': 'nx_a1b2c3d4...' }
});
const config = await res.json();
console.log(config.prefix); // '!'
PATCH /api/data

อัปเดตหรือแก้ไขค่าเฉพาะบางส่วน (ไม่ต้องส่งไปทั้งก้อน)

Body (JSON)

{
  "status": "online",
  "usersCount": 150
}

* ระบบจะเอาค่าใหม่ไปทับค่าเก่า และเก็บค่าอื่นๆ ที่ไม่ได้ส่งมาไว้เหมือนเดิม
